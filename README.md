## Tip
```c++
Avltree avltree;

int key;
int value;

avltree.Insert(key, value);
avltree.Search(key);
avltree.Delete(key);

```

```c++
Btree * btree;

int order = 200;
btree = new Btree(order);

int key;
int value;

btree->Insert(key, value);
btree->Search(key);
btree->Delete(key);

```
